# Project OSAI E-commerce platform

Project OSAI is an e-commerce web application built using React, Node.js, MySQL, and Bootstrap. It allows customers to log in, browse products, add and checkout to a basket, place orders and check order history. Admins can log in and manage inventory, update stock levels, process orders, manage customer details on top of being able to use the site as a regular customer.
